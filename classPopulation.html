Member Function Documentation

NeuralNet * Population::BestNet ( )

Returns the best neural net in the population.

Uses the fitness value calculated by the most recent call to Evaluate or Reproduce. If neither has been called, returns nullptr.

unsigned int Population::NumSpecies ( )

Returns the number of species in the population.

Uses the speciation from the most recent call to Reproduce. If Reproduce has not been called, returns 0.
